4.12.5. документа "The java language specification, Java SE 17 Edition"

В этом разделе описаны правила, по которым переменные инициализируются начальными значениями

Согласно этому разделу локальной переменной должно быть явно присвоено значение перед использованием. Поэтому локальные
переменные не имеют значения по умолчанию. Это позволяет исключить варианты C и D
